# Labs

Labs are small, practical exercises that teach system design through artifacts, not just reading.

The goal is not to build impressive demos. The goal is to expose the design decisions that make AI systems work or fail in production.

## Current Labs

1. [RAG Retrieval Eval Lab](./rag-retrieval-eval/README.md)

## Lab Standard

Each lab should include:

- A system design lesson
- A runnable or inspectable artifact
- A small dataset or scenario
- Expected outputs
- Failure analysis questions
- Extension tasks

Labs should work with minimal setup whenever possible.
